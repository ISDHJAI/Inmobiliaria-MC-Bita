# Bita — MC Inmobiliaria

Landing page del modelo **Bita** en Privada Sotavento, Villa de Pozos, San Luis Potosí.

- Diseño minimalista en morado ciruela de MC Inmobiliaria (`#7b366e` / `#5e2558`).
- HTML + CSS (`css/styles.css`) + JS (`js/main.js`), sin dependencias.
- Botones de WhatsApp (52 444 411 9732) con mensaje precargado y botón flotante.
- Galería (27 fotos) y amenidades (13 fotos) con carrusel y lightbox.

- Video del recorrido en `videos/reel-bita.mp4` (comprimido a 38 MB para web).
